Based on the Consumption Value Model, this project constructed a theoretical framework for college students' willingness to pay for knowledge products.

**Key Contributions:**
- Designed a Likert-scale questionnaire and collected 342 valid responses
- Applied SPSS and Structural Equation Modeling (Amos) for reliability/validity testing and path analysis
- Found that social value positively influences payment willingness, while emotional value exerts a negative effect
- Responsible for writing the data analysis section; project successfully completed
